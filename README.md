Banana Chocolate Chip (Muffin)
==============================

A framework for modern web apps based on [Muffin](http://richbray.me/muffin/)

##INSTALL

**Important make sure you've installed Ruby on Rails & Nodejs**

Install the dependancies

```
npm install
```


Run gulp

```
gulp
```

That's it.
